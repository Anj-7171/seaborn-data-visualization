# Exploring Seaborn Data Visualization

A beginner-friendly project demonstrating the most commonly used **Seaborn** visualization techniques for **Exploratory Data Analysis (EDA)**. This repository showcases various statistical plots using Seaborn's built-in **Tips** dataset and explains their purpose, implementation, and interpretation.

Whether you're new to data visualization or looking for a quick reference to Seaborn plots, this project provides practical examples along with detailed explanations.

---

## Project Overview

Data visualization is an essential step in understanding datasets before performing statistical analysis or building machine learning models. Seaborn simplifies this process by providing high-level functions for creating informative and visually appealing statistical graphics.

This project explores a variety of Seaborn plots commonly used during Exploratory Data Analysis and demonstrates how each visualization helps uncover patterns, trends, relationships, and distributions within data.

---

## Visualizations Covered

- Scatter Plot
- Relational Plot (Hue = Day)
- Relational Plot (Hue = Time)
- Faceted Relational Plot
- Bar Plot
- Count Plot
- Box Plot
- Distribution Plot (Histogram + KDE)
- Joint Plot
- Pair Plot
- Correlation Heatmap
- Annotated Heatmap

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Anj-7171/seaborn-data-visualization.git
```

Navigate to the project directory:

```bash
cd seaborn-data-visualization
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open the notebook located in the `notebook/` directory and run all cells to reproduce the visualizations.
